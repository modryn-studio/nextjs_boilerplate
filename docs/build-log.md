# Modryn Studio — Build Velocity Log

_Track Luke Hours only — not AI generation time. Not calendar time. Not waiting._

---

## How to Use This File

**One entry per session. Four fields. That's it.**

Copy this block and fill it in at the end of every session:

```
### Session [number] — [Project Name]
- **Date:** YYYY-MM-DD
- **Session Type:** Advisor Session / AI OS Build Session / Decision Session / Integration Session
- **Luke Hours:** X.X
- **What Shipped or Decided:** One sentence. What moved forward that didn't exist before this session.
```

**What counts as Luke Hours:**

- Your judgment is actively in the loop
- You are reading, reviewing, deciding, directing
- You are in an advisor session like this one

**What does NOT count:**

- AI generation running while you're away from the keyboard
- Build time without your active direction
- Sleep, waiting on dependencies, calendar time

---

## Session Types

| Type                    | When to use                                                                        |
| ----------------------- | ---------------------------------------------------------------------------------- |
| **Advisor Session**     | Time spent with the strategic advisor — planning, reviewing, directing             |
| **AI OS Build Session** | Active direction of the AI OS team — briefing, reviewing, approving or redirecting |
| **Decision Session**    | Resolving a named dependency or open question blocking progress                    |
| **Integration Session** | Assembling, testing, reviewing, and shipping output                                |

---

## The Three Numbers (calculate at project close)

| Number              | What it is                         | How to calculate                   |
| ------------------- | ---------------------------------- | ---------------------------------- |
| **Luke Hours**      | Your active time only              | Sum of all Luke Hours below        |
| **Wall Clock Days** | Calendar days greenlight → shipped | Shipped date minus greenlight date |
| **Leverage Ratio**  | How much output per Luke hour      | AI OS estimate ÷ Luke Hours actual |
